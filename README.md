# Space Warp

A space warp experience inspired by [Felix Tan project](https://github.com/o2bomb/space-warp)

## [TRY IT](https://mathieu-superpose.github.io/space-warp)

## Technology

- Bundler: Vite
- JS library: React
- 3d renderer: R3F
- R3F helpers: Drei
- Post processing: react-postprocessing
- Deployment: gh-pages
